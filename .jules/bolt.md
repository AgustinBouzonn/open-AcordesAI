## 2024-05-24 - Optimizing React Animation Loops
**Learning:** `setInterval`-based animations in React `useEffect` hooks create performance bottlenecks due to lack of synchronization with the display's refresh rate and cause jank. By migrating to `requestAnimationFrame` with a time-delta calculation (`delta = time - lastTime`) and using `useRef` for variable tracking (e.g., `scrollAccumulator`), the scrolling becomes frame-rate independent. This guarantees smooth motion and drastically improves battery and CPU efficiency while preventing animation loop resets when closures capture state.
**Action:** Always prefer `requestAnimationFrame` and a time-delta calculation over `setInterval` for any continuous UI animations in React components, and rely on `useRef` to maintain values across frames without triggering unnecessary re-renders or loop restarts.
