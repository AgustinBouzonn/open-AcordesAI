## 2024-05-18 - Added ARIA labels and focus rings to icon-only buttons
**Learning:** Icon-only buttons (like Favorite, Comments, Font Size controls) in the SongViewer component were missing basic accessibility attributes. Without `aria-label`, screen readers would not announce their purpose. Additionally, adding `focus-visible:ring-2 focus-visible:ring-brand focus-visible:outline-none` greatly improves keyboard navigation visibility, and adding `aria-hidden="true"` to the `lucide-react` SVGs prevents redundant screen reader announcements.
**Action:** Always ensure icon-only interactive elements have descriptive `aria-label` attributes (in the app's primary language, Spanish), `title` attributes for tooltips, and clear `focus-visible` styles for keyboard accessibility. Add `aria-hidden="true"` to the icons themselves.
