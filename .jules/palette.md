## 2024-03-20 - Accessible Icon-Only Buttons
**Learning:** This app frequently relies on icon-only buttons for interactive features (e.g., adding to favorites, opening comments, adjusting font size, auto-scrolling) using standard components like Lucide React icons. However, by default these lack necessary accessible names, semantic button types, screen reader decorators for the inner decorative icon, and visible focus states for keyboard users.
**Action:** When adding or modifying interactive controls with icons in the AcordesAI app, always ensure they possess an explicit `type="button"`, descriptive Spanish localization using both `aria-label` and `title` attributes (e.g., "Añadir a favoritos"), keyboard focus styling (`focus-visible:ring-2 focus-visible:ring-brand focus-visible:outline-none`), and `aria-hidden="true"` applied to the decorative inner icon element to prevent screen reader redundancy.
