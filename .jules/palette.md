## 2024-04-15 - Dynamic ARIA attributes on stateful icon-only buttons
**Learning:** In the SongViewer component, icon-only buttons like the Favorite and Comments toggle had no text label for screen readers or standard tooltips. The action these buttons perform changes based on the state (e.g., "Añadir a favoritos" vs "Quitar de favoritos", "Mostrar opiniones" vs "Ocultar opiniones"). Providing a static `aria-label` is insufficient because the button's action changes.
**Action:** Always ensure that stateful icon-only buttons use a dynamic `aria-label` and `title` that reflect the *action* that will happen or the current state. Apply `aria-hidden="true"` to the decorative icon SVG, and add `focus-visible` classes (e.g., `focus-visible:ring-2 focus-visible:ring-brand focus-visible:outline-none`) to ensure keyboard users can clearly see when they have focused the button.
