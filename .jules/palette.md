## 2024-03-10 - Spanish Screen Reader Accessibility for Icon-only Buttons
**Learning:** Icon-only buttons (like those used for Favorites, Comments, and Font Size adjustments) lack context for visually impaired users relying on screen readers. Adding explicit localized labels (`aria-label`) and tooltips (`title`) alongside explicit `type="button"` and focus rings (`focus-visible`) is essential for keyboard navigation and screen-reader accessibility, particularly in Spanish-language interfaces where these additions greatly improve usability and meet basic accessibility requirements.
**Action:** Always verify that every interactive icon-only button includes an explicitly localized `aria-label`, a `title` for tooltip display on hover, a clear visible focus ring for keyboard navigation, and the `type="button"` attribute to prevent unintended form submissions.
